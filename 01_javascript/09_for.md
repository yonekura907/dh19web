# forループ

反復作業を即時に実行する


### for文の記法

```
for (初期化; 条件; 更新 ){
	文
}

```
&nbsp;

```
for(var i=0; i < 10; i++){
	console.log(i);
}

```
0から10未満（9）まで繰り返す

&nbsp;
&nbsp;

## 配列とfor文

```
var fruit = ['Apple','Orange','Grape','Strawberry'];

for(var i=0; i < fruit.length; i++){
	document.write(fruit[i]);
}

```
