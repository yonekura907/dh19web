
# オブジェクト

`key`と`value`の形式で複数の値をデータとして保存することができる


```
var omikuji = [
    {
        result: '大吉',
        love: '叶う',
        business: '勝負せよ'
    },{
        result: '吉',
        love: '出会う',
        business: '現状維持'
    },{
        result: '凶',
        love: '出会わず',
        business: '局面を迎える'
    }
];

```
